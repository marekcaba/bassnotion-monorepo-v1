### Prerequisites

- Node.js v20.x

### Initial Setup

```bash
# Install dependencies
pnpm install

# Build all packages in correct order
pnpm nx run-many --target=build --projects=@bassnotion/contracts,@bassnotion/backend,@bassnotion/frontend --parallel=false
```

## đź› ď¸Ź Development Mode

### Starting Services with PM2

```bash
# Start backend on port 3000 and frontend on port 3001
pm2 delete all  # Clean up any existing processes
cd apps/backend && PORT=3000 pm2 start "node ../../dist/apps/backend/src/main.js" --name "bassnotion-backend"
cd ../frontend && PORT=3001 pm2 start "pnpm dev" --name "bassnotion-frontend"
```

### Important Path Information

- Backend build output: `dist/apps/backend/src/main.js`
- Frontend dev server: Runs from source directly
- Contracts build output: `dist/libs/contracts`


- Frontend: Automatic with Next.js dev server
