## 📝 Documentation Guidelines

When adding new documentation:

1. **Use Clear Titles** - Make it obvious what the document covers
2. **Add to Correct Folder** - Developer docs go in `developer-handbook/`
3. **Include Examples** - Show, don't just tell
4. **Update This README** - Add your new doc to the appropriate section
5. **Date Your Docs** - Add "Last updated" timestamp at the bottom
